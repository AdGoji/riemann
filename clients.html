Clients send events to Riemann.

Riemann clients speak Protocol Buffers over TCP and UDP. They can submit events to Riemann and query the index.

Tools

A Ruby gem with little daemons to monitor key systems.

Ruby

The Riemann Ruby client is my threadsafe, auto-retrying reference implementation, supporting both TCP and UDP. It also provides utility classes to help manage service state.

Clojure

Riemann comes with a built-in Clojure client. It's thread-safe, stateful, and synchronized. Does not support UDP yet. You can require it as a dependency from Clojars. See riemann-bench for an example application using the client.
