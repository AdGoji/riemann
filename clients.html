Clients send events to Reimann.

Reimann clients speak Protocol Buffers over TCP and UDP. They can submit events to Reimann and query the index.

Ruby

The Reimann Ruby client is my threadsafe, auto-retrying reference implementation, supporting both TCP and UDP. It also provides utility classes to help manage service state.

Clojure

Reimann comes with a built-in Clojure client. It's thread-safe, stateful, and synchronized. Does not support UDP yet. You can require it as a dependency from Clojars. See reimann-bench for an example application using the client.
