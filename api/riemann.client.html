close-client
(close-client client)
Close a client.

query
(query client string)
Query the server for events in the index. Returns a list of events.

reconnect-client
(reconnect-client client)
Reconnect a client.

send-event
(send-event client event)(send-event client event ack)

tcp-client
(tcp-client & {:keys [host port], :or {port 5555, host "localhost"}, :as opts})
Create a new TCP client. Example:

(tcp-client)
(tcp-client :host "foo" :port 5555)
udp-client
(udp-client & {:keys [host port], :or {port 5555, host "localhost"}, :as opts})

(udp-client)
