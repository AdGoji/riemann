riemann.query documentation

ast
(ast string)
The expression AST for a given string

fun
(fun ast)
Transforms an AST into a fn [event] which returns true if the query matches
that event. Example:

(def q (fun (ast "metric > 2")))
(q {:metric 1}) => false
(q {:metric 3}) => true
