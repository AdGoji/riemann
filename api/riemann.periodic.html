riemann.periodic documentation

after
(after f)(after delay f)
Calls f after interval seconds.

deferrable-every
(deferrable-every interval delay f)
Calls f every interval seconds, until stopped. Returns a control function.
Can be deferred. I'll probably change this API.
every
(every f)(every interval f)(every interval delay f)
Calls f every interval seconds, until stopped. Returns a fn which stops.
