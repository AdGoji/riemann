Reimann monitors distributed systems.

Reimann aggregates events from your servers and applications with a powerful stream processing language. Send an email for every exception raised by your code. Track the latency distribution of your web app. See the top processes on any host, by memory and CPU. Combine statistics from every node in a riak cluster and forward to Graphite. Send alerts when a key process fails to check in. Know how many users signed up right this second. Reimann provides low-latency, transient shared state for heteogenous systems.

Powerful stream primitives

(where (and (service "api") 
              (not (tagged "transient")))
    graph
    index
    (changed-state 
      (email "5551231234@vtext.com")))

Query language

state = "error rate" and (not host =~ "www.%")

Search the Reimann index with a small query language. Clients can monitor each other, generate reports, or produce dashboards.
